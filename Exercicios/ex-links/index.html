Abrir um link em uma nova aba tem que ter o target="_blank" dentro do parametro <a>

            
Exemplo de Código
                
Código para abrir um link em uma nova aba 

        
            <a href="ENDEREÇO" target="_blank" rel=external" hreflang="pt-br">
target="_blank" indica que vamos abrir uma nova aba
rel="external" relativo = usamos semantica no html
hreflang para avisar os mecanismo de tradução como lidar com o site que vamos abrir. Qual linga ele vai se deparar né, isso facilita e torna o processo de user-experience mais agradável

Uso de Links Internos

Você pode usar links internos também e que não tem necessidade de usar o parametro target= blank, mas se quiser usar, deveria ser _self ao invés de _blank.
